Sentiment News Prototype

This project involved the development of a prototype application for sentiment analysis in news articles, designed to classify content extracted from multiple sources using web scraping techniques and neural networks. The goal was to provide an intelligent tool capable of identifying the tone of news stories—positive, negative, or neutral—and presenting the analysis in a clear, visual format.

I was responsible for the entire development lifecycle of the application, from data extraction to model integration and UI design. The project included building web scraping pipelines, implementing and training a neural network for sentiment classification, and creating a user interface to visualize the results.
